# Next Project Community

Welcome to the official documentation for FiveM plugins developed by **Next Project Community**.

:::caution 
Available on Tebex
All plugins are available for purchase on **[next-project.tebex.io](https://next-project.tebex.io/)**  
After purchase, download them from your CFX portal at **[portal.cfx.re](https://portal.cfx.re)**.
:::

## Available plugins

| Plugin | Description |
|--------|-------------|
| [Next Deliveries](next-deliveries/intro) | Multi-framework delivery company system |
| [Next Gas Stations](next-gas-stations/intro) | Multi-framework buyable gas station system |

## Supported frameworks

All plugins support the following frameworks:

- **Qbox / QboxCore**
- **QBCore**
- **ESX**

## Common requirements

- [`ox_lib`](https://github.com/overextended/ox_lib)
- [`oxmysql`](https://github.com/overextended/oxmysql)
