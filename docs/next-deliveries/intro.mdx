# Introduction

**Next Deliveries** is a multi-framework delivery company system for FiveM.  
Players can create their own company, hire employees, and complete truck delivery missions.

## Features

- Create and manage delivery companies
- Delivery missions with multiple categories (`fuel`, `goods`)
- Multiple vehicle types (small, van, medium, large, XL)
- Rewards credited to the company account
- Banking integration (`qb-banking`, `renewed-banking`, `esx_addonaccount`)
- Garage integration (`qb-garages`, `qbx_garages`, `esx_garage`)
- Server API to inject missions from other resources
- Compatible with **Qbox**, **QBCore**, **ESX**
